About Me

Hello! I'm Jose Antonio, a passionate Game Engine Developer and Gameplay Programmer with a deep understanding of game architecture and systems programming.

With expertise in low-level graphics programming and game physics, I specialize in creating robust game mechanics, intelligent AI systems, and optimized rendering pipelines. My technical background allows me to bridge the gap between creative vision and technical implementation.

I have extensive experience working with multiple programming languages and technologies, including:

  • C/C++ for performance-critical systems and engine development
  • C# for rapid prototyping and Unity development
  • Lua for game scripting and extensibility
  • Graphics APIs including OpenGL and Vulkan
  • Game Engines like Unity and Godot

My passion lies in crafting elegant solutions to complex problems and creating immersive interactive experiences. I'm constantly exploring new technologies and techniques to push the boundaries of what's possible in game development.
