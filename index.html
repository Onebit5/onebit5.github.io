About Me

Hello! I'm Jose Antonio, a game developer specializing in both engine architecture and gameplay programming. Currently contributing to the Godot Engine open-source project, I bring hands-on experience building game systems from the ground up.

I've developed OneTek Engine 2, a cross-platform 3D engine featuring multi-API rendering (Vulkan/DirectX 12/OpenGL), PhysX integration, and AI navigation systems. My work spans from low-level engine architecture to shipping complete games like Cosmic Chaos on Itch.io.

My technical expertise includes:

  • Graphics Programming Vulkan, DirectX 12, OpenGL, PBR materials, shader programming
  • Gameplay Systems AI behavior, animation systems, multiplayer networking
  • Languages & Tools C/C++, C#, Unity, Godot, CMake, Git

I'm passionate about building efficient, maintainable systems that enable creative expression while maintaining technical excellence.
